# Ralph Starter Template

A ready-to-use template for the Ralph Wiggum AI coding workflow. Build products faster with autonomous AI development loops.

Based on [Ryan Carson's Ralph methodology](https://github.com/snarktank/ralph).

## What's Included

| File | Purpose |
|------|---------|
| `setup.sh` | One-command session startup |
| `scripts/ralph/ralph.sh` | Autonomous development loop |
| `scripts/ralph/prompt.md` | Instructions for each Ralph iteration |
| `scripts/ralph/ideate.md` | Creative brainstorming framework |
| `prd.json` | User story tracker |
| `progress.txt` | Session learnings log |
| `CLAUDE.md` | Project context for Claude |

## Quick Start

1. Import this template to Replit
2. Update `CLAUDE.md` with your project details
3. Run `./setup.sh` to start
